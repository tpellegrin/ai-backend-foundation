# Task T-1100 — `app/ai_governance/{domain,ports}.py` (pre-llm interface only)

> Section: **S11 — LLM + embeddings ports (with fakes and contract tests)**
>
> Before you start, read [`../rules.md`](../rules.md) (global rules, stop signals, allowed-files discipline, command discipline, lower-complexity model instructions).
> After you finish, self-check against [`../review.md`](../review.md) (final review checklist, architecture compliance).

---


Purpose

- Land the **interface surface** of `ai_governance` (domain types + outbound persistence ports) so S12 can implement the governance service against a stable contract. The persistence/service/api/wiring of governance still live in S12. The `GovernanceGate` Protocol consumed by `LlmService` is **not** defined here — per [ADR-0024](../../adr/0024-define-llm-owned-governance-gate-contract.md) it is owned by `app.llm.ports` (T-1101); this task must not force `app.llm` to import from `app.ai_governance`.

Depends on: T-201, T-203.

Allowed files: `app/ai_governance/__init__.py`, `app/ai_governance/domain.py`, `app/ai_governance/ports.py`, `app/ai_governance/tests/test_domain.py`, `app/ai_governance/tests/test_ports.py`.

Forbidden

- No FastAPI, SQLAlchemy, provider SDKs, or `app.infrastructure.*` imports.
- Do **not** implement `service.py`, `persistence.py`, `events.py`, or `api.py` here — those land in S12 (T-1202..T-1205).
- Do not depend on `app.llm.*` or `app.embeddings.*`.

Implementation requirements

- `domain.py`: `BudgetPolicy(tenant_id, monthly_usd: Decimal, warning_threshold: float, model_allowlist: tuple[str, ...])`; `UsageEntry(tenant_id, model, tokens_in, tokens_out, cost_usd, occurred_at, request_id)`. `AllowDecision`, `BudgetExceededError`, and `ModelNotAllowedError` are **not** defined here — they live in `app.shared.governance` / `app.shared.errors` (see [ADR-0024](../../adr/0024-define-llm-owned-governance-gate-contract.md)) because they cross the `app.llm` ↔ `app.ai_governance` boundary and must be importable by a lower layer. Import them from `app.shared` where needed inside this module.
- `ports.py`: `UsageRepository(Protocol)` async `record(entry) -> None`, `monthly_usage_usd(tenant_id) -> Decimal`; `BudgetPolicyStore(Protocol)` async `get(tenant_id) -> BudgetPolicy`. **Do not** define `GovernanceGate` here; that Protocol is owned by `app.llm.ports` (T-1101). This module may optionally re-export a same-layer `GovernanceGate` alias in a later S12 task if `app.ai` or `app.rag` grow a need for it, but S11 does not require it.
- `__init__.py` re-exports **only** `BudgetPolicy`, `UsageEntry`, `UsageRepository`, `BudgetPolicyStore`. `AllowDecision` / `BudgetExceededError` / `ModelNotAllowedError` are re-exported from `app.shared`, not from `app.ai_governance`.

Tests required

- Dataclass invariants (immutability, value ranges) for `BudgetPolicy` and `UsageEntry`.
- Protocol satisfiability of `UsageRepository` and `BudgetPolicyStore` against in-test fakes.
- (`AllowDecision` construction tests and `GovernanceGate` Protocol satisfiability tests live with their owners: `app.shared` tests for `AllowDecision`, `app.llm` tests for `GovernanceGate`.)

Acceptance criteria

- `app.llm` does **not** import from `app.ai_governance` (per [ADR-0024](../../adr/0024-define-llm-owned-governance-gate-contract.md)). This task must not add any symbol that would tempt or require such an import.
- `lint-imports` remains clean (in particular, the `Layers` contract is not weakened and no `ignore_imports` entry is introduced).

Commands

```
make fmt
make lint
make typecheck
make test
```

Common failure modes

- Importing SQLAlchemy or FastAPI here.
- Implementing `service.py` or `persistence.py` prematurely.
- Forgetting to re-export the ports through `__init__.py`.

Review checklist

- Pure domain + Protocols only; no I/O imports; `__init__.py` is the public surface; `lint-imports` green.
